Como usar no GitHub Pages

Apenas o site (zyllah_v6.html renomeado para index.html) vai para o repositório público zyllahdigital.github.io. Todos os outros arquivos são de uso interno — mantenha-os localmente ou em uma pasta privada. Para subir o site: renomeie o arquivo para index.html, faça upload no repositório e aguarde 1–2 minutos para publicar em https://zyllahdigital.github.io.
